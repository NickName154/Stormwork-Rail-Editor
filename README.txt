## Getting started ##
An example mesh is given
Make sure your export mesh only has vertexes and edges
Export your track mesh as a ASCII .ply file

Import it into the tool

## Junction Manipulation ##
Clicking a junction (blue node) then click on any adjacent node to set it as the TURN node (green).

## Merging ## 
Clicking the "M" of the source tile and then the destination tile merges the tiles together so they export as one with the same origin.
useful for when you have part of tracks outside of the tile boundary!

## Export ##
Clicking the "E" will prompt you to chose a file, Select your tile.xml you wish to put the track layout onto.

## Cleaning ##
Clicking "Clear tile.xml" will prompt you to chose your tile, which will then clear out any track node in it.

Always and i do mean ALWAYS make a backup of your tiles before using this tool!
